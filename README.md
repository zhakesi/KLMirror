# KLMirror
本项目可作为开发视频图像滤镜特效的示例项目。系统:Android, IDE: Android Studio, Graphic API:OpenGL ES3.0, 特效滤镜包括Anime4K, 3D Lut, Gussian Blur等。<br/>
